# MyGoalAI

Привет!
Я выкладываю один из своих проектов в загажнике в open-source. Рабочее название енного творения - MyGoalAI.
Сей проект был сделан исключительно для того, чтобы подключить туда нейронку и идти к своим целям быстрее.
В данный момент тут есть:
- создание/удаление целей
- добавление/удаление шагов для достижения целей
Если кто-то захочет разобраться в нем - Welcome :)
